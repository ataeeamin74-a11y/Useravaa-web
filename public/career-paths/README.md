# Useravaa Career Path Visual Assets

Each folder in this directory maps to one current `/career/paths/{slug}` SEO page.

The folder path is:

```text
public/career-paths/{slug}/
```

Each slug folder should later contain exactly these manually supplied image assets:

```text
hero-mascot.png
fit.png
job-reality.png
difficulties.png
ai-impact.png
interview-questions.png
```

These files are intentionally not generated by code. Do not add fake images, blank PNGs, stock artwork, or generated placeholders here. The app consumes the files only when the manually prepared assets exist, and keeps a styled in-app placeholder when they are missing.
