# Access & Permission Rules

| Action | Provider | Requester | Admin |
|---|---:|---:|---:|
| View own weekly question | yes | no | no |
| Replace own question | yes | no | no |
| Skip own question | yes | no | no |
| Draft own answer | yes | no | no |
| Publish own answer | yes | no | no |
| Retract own answer | yes | no | no |
| View published answers | yes | yes | yes |
| Review answers | no | no | no |
| Approve/reject/hide | no | no | no |
