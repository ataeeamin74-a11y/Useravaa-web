# Functional Specification

## Modules
1. Weekly Question Generation
2. Provider Question Card
3. Answer Drafting
4. Provider Responsibility Confirmation
5. Direct Profile Publication
6. Provider Retraction
7. Public Profile Display
8. Analytics Tracking

## Answer Submission
- Provider can save draft.
- Provider must explicitly confirm responsibility before publication.
- After confirmation, answer becomes `published`.
- There is no Admin review, approve, reject or hide flow in MVP.
