# KPI Tracking Plan

- Weekly answer rate = published answers / questions shown
- Question replacement rate = question_replaced / questions_shown
- Skip rate = question_skipped / questions_shown
- Published answer rate = answers_published / answer_drafts_created
- Retract rate = answers_retracted / answers_published
- Profile view to request rate = requests_from_profile / profile_views
- Save rate = saves_after_profile_view / profile_views
