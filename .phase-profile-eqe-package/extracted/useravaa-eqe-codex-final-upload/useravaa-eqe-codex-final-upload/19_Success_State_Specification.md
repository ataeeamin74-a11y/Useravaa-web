# Success State Specification

| Action | Success |
|---|---|
| Save draft | پیش‌نویس ذخیره شد. |
| Publish answer | پاسخ در پروفایل منتشر شد. |
| Replace question | سؤال جدید نمایش داده شد. |
| Skip question | تا هفته بعد سؤال جدیدی نمایش داده نمی‌شود. |
| Retract answer | پاسخ از پروفایل حذف شد. |
