# Error Message Specification

| errorCode | Message |
|---|---|
| NO_ELIGIBLE_TEMPLATE | فعلاً پرسشی برای پروفایل شما آماده نیست. |
| NO_REPLACEMENT_AVAILABLE | سؤال جایگزین دیگری برای این هفته وجود ندارد. |
| INVALID_QUESTION_STATE | وضعیت این پرسش اجازه انجام این کار را نمی‌دهد. |
| ANSWER_REQUIRED | پاسخ را بنویسید. |
| ANSWER_TOO_LONG | پاسخ نباید بیشتر از ۷۰۰ کاراکتر باشد. |
| ANSWER_TOO_SHORT | حداقل یک جمله معنادار بنویسید. |
| RESPONSIBILITY_NOT_ACCEPTED | برای انتشار باید مسئولیت محتوا را تأیید کنید. |
| RETRACT_NOT_ALLOWED | امکان حذف این پاسخ از پروفایل وجود ندارد. |
