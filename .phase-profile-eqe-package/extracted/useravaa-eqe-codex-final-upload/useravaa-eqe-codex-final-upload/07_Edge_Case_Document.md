# Edge Case Document

| Edge Case | Behavior |
|---|---|
| Required data missing | Template excluded |
| No eligible template | Do not show card; log NO_ELIGIBLE_TEMPLATE |
| Replacement unavailable | Disable replace or show message |
| Skip | Suppress until next week |
| Answer > 700 chars | Block publish |
| No responsibility confirmation | Block publish |
| No published answers | Hide «از تجربه من» |
| More than 3 published answers | Show latest 3 |
| Provider retracts answer | Remove from public profile |
