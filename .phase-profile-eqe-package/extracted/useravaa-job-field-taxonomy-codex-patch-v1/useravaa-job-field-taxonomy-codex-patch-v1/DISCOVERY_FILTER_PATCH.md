# Discovery Filter Patch

## Rule

Discovery job field filter must use only the fixed taxonomy.

## Filter Label

حوزه شغلی

## Filter Values

- محصول و تجربه کاربر
- طراحی گرافیک و هویت بصری
- فنی و مهندسی نرم‌افزار
- علوم داده و هوش مصنوعی
- مارکتینگ و برند
- تحلیل و توسعه کسب‌وکار
- عملیات
- تجربه مشتری
- پشتیبانی مشتریان
- فروش و بازرگانی
- استراتژی و مدل کسب‌وکار
- مالی، حقوقی و سرمایه‌گذاری
- منابع انسانی و فرهنگ سازمانی
- مدیریت، رهبری و کارآفرینی

## Remove Old Filter Values

Do not use old labels as standalone taxonomy:
- محصول
- هوش تجاری
- تحلیل داده
- طراحی محصول
- رشد
- مهندسی

Map old fixture data into the new taxonomy before rendering.
