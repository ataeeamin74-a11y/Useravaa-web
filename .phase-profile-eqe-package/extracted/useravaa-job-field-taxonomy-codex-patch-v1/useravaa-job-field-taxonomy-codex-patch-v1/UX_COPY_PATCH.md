# UX Copy Patch

## Field Labels

عنوان شغلی  
حوزه شغلی

## Helper Text

### عنوان شغلی

عنوان شغلی را همان‌طور که در کار واقعی استفاده می‌کنید وارد کنید.

### حوزه شغلی

حوزه شغلی را از لیست انتخاب کنید. این دسته‌بندی برای جستجو، فیلترها و پرسش‌های تجربه استفاده می‌شود.

## Validation Messages

### Invalid Job Title

عنوان شغلی را کامل وارد کنید.

### Invalid Job Field

حوزه شغلی را از لیست انتخاب کنید.

## Timeline Item Copy

در هر سابقه، عنوان شغلی را دستی وارد کنید و حوزه شغلی را از لیست ثابت انتخاب کنید.
